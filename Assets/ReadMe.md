# 折纸DEMO
## 简介
> 这是一个以纸为载体的2D平台跳跃模板，纸可以随意的折叠和复原。
## 操作指南
> 1. 左键拖拽确定背面移动范围
> 2. 左键放开确认折叠操作
> 3. 上方按键可以翻页和复原
## 实现概览
> 1. 将正面和背面分成两块，鼠标点击时根据鼠标位置生成背面的副本用作显示，并且保存当前状态用作撤销
> 2. 鼠标拖动的时候同时拉动遮罩和背面
> 3. 确认折叠之后修改碰撞体重新渲染正面和背面
## 仍需改进的部分
> 1. 方便起见只实现了水平方向折叠的功能，但不难将其扩展至任意方向
> 2. 可以指定需要复原的折痕而非仅仅撤销上次操作
